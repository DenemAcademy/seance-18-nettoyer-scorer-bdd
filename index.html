PHASE 3 — CONSOLIDATION DES DONNÉES
Identifie les fichiers qui contiennent vraiment des leads.

Règles de lecture :

- téléphone le plus exploitable

Tout passer dans l'outil de prospection parce que le fichier existe.

Choisir `final.xlsx` juste parce que le nom rassure.

Lire les colonnes, onglets, volumes et sources avant de choisir.
